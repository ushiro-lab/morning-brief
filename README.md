# Morning Brief

朝のニュースを選び、音声ブリーフにまとめる、費用をかけないWebアプリです。

`index.html` をスマホまたはPCのブラウザで開くと動作します。毎朝7:00にSmartNewsで「国内経済」と「山口県」の記事を確認し、選んだ見出しを最大5本貼り付けます。アプリが落ち着いた解説原稿を作り、ブラウザ内蔵の日本語読み上げで再生します。設定・記事履歴はブラウザのローカルストレージに保存されます。

スマホでホーム画面に追加してアプリのように起動するには、HTTPSで公開したURLが必要です。一度開いた画面はオフラインでも表示できます（RSS取得とSmartNewsの閲覧には通信が必要です）。ローカルの `index.html` を直接開く場合は、通常のブラウザページとして利用できます。

## 無料で使うための制約

1. 本アプリは外部の有料ニュースAPI・AI API・音声APIを使用しません。
2. SmartNewsは記事閲覧用のサービスであり、利用者向けの取得用RSSは用意されていないため、記事の見出しは手動で貼り付けます。山口県公式の報道発表RSSは初期設定済みです。
3. ブラウザは閉じていると自動生成・自動再生できません。毎朝の完全自動配信には、常時稼働する端末またはサーバーが必要です。

費用を発生させない方針のため、APIキーを必要とする機能は追加していません。
